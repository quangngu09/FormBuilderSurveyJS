## Package

survey-library / survey-core: core của thư viện survey js
<br />
survey-library / survey-react-ui: thư viện ui cho react


survey-creator / survey-creator-core: core của phần creator ( builder )
<br />


## Install

init library first
```
cd survey-library
npm run init
npm run build
```

init creator
```
cd survey-creator
npm run init
npm run build
```

## Test

```
cd survey-test
npm i
npm start
```
